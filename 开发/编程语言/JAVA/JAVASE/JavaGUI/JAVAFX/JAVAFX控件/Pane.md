# BorderPane
提供了5个放置节点的区域：top, bottom, left, right, 和 center

# HBox
HBox横的一个一个摆

# VBox
VBox竖的一个一个摆

# StackPane
StackPane后来居上，新放的东西会覆盖原来的东西

# GridPane
GridPane一格一格的放，可以设置行和列

# FlowPane
FlowPane会一行一行的摆，放不下就拐到下一行

# SplitPane
SplitPane能用鼠标拖动的面板

# Accordion
Accordion可以翻的页面

# TabPane
TabPane一个一个的标签

# AnchorPane
跟随外面的移动，外面的往左，里面的就往左，外面的往右，里面的就往右
